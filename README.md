## 谷歌语法

可利用谷歌语法搜索社工库

**inurl：**

用于搜索包含的url关键词的网页，如：inurl: admin.php

**intitle** 

搜索网页标题中的关键字，如：intitle: 久久

**intext：**

搜索网页正文中的关键字，如：intext: 2022世界杯

**filetype：**

按指定文件类型即文件后缀名搜索，如：filetype: xls 身份证
